## Instalacion:

la instalacion debe ser como usuario ROOT.

  ```sh
  chmod +x install.sh
  ./install.sh
  ```

Una vez ejecutado el instalador automatico se podra usar la herramienta con:
  ```sh
  python3 main.py
  ```
